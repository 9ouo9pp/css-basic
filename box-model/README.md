# 박스 모델

## 박스 모델의 기본 구성

- 콘텐 영역
- 패딩 (padding)
- 테두리 (border)
- 마진 (margin)

## 박스 모델의 크기를 계산하는 box-sizing 속성

- content-box 기본값
- border-box 테두리까지 포함해서 너빗값을 지정

## 박스 모델에 그림자 효과를 주는 box-shadow 속성

- box-shadow: inset 수평거리 수직거리 흐림정도blur 번짐정도spread 색상

- 피그마에서 Effects > Drop shadow, Inner shadow에 해당
